    集成Springboot+security，包含正常表单登录、ajax登录、自定义验证、方法安全等功能；
    遇到了一些坑或者可扩展在代码里有详细注释。。
    注：因仅用于后台功能demo的实现，所以前台页面跳转没有实现，需手动输入接口地址进行权限测试。。。。
